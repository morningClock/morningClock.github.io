hexo server一直挂起无法启动解决方法

这几天用hexo捣鼓博客吧，遇到了一个很奇怪的问题。就是hexo server启动本地服务器一直处理starting processing。

百度搜了一圈都没找到想要的结果，去hexo-server的issue逛了一圈，发现还是有收获的。

https://github.com/hexojs/hexo-server/issues/124

这位老哥和我情况几乎一样

后来发现是页面文件夹中多出了一些无效的符号链接文件。

开始吧，我还是不大懂这问题出在哪里。

于是我就重新init了一个blog，把配置和文件分别迁移过去，看看启动还是不是有问题，利用排除法来看问题出在哪里了。

于是就在我复制source文件夹下所有文章过去的时候，提示说有很多文件出现找不到路径的问题，无法复制和删除。

这些就是无效符号链接的文件，到指定目录看，都傻眼了，文章对应的img文件夹，竟然是个无限嵌套的文件夹。无法使用强制删除，于是折中方法就是先丢进回收站，然后再清空回收站，提示错误再清你就会发现清空成功了,至于是哪里导致的毛病，由于无法复现问题，所以暂时还没办法定位。

总结一下解决方法

找到产生的无效文件删除，就可以正常使用hexo-server了。
