CSS 清除浮动方法汇总

1.增加空div浮动盒子

增加一个空div在浮动元素后面，并设置clear:both属性

<div>
    <!-- 浮动元素 -->
    <img class="img-left" src="">
    <!-- 清除浮动clear:both -->
    <div style="clear: both;"></div>
</div>

2.overflow

在浮动元素的父元素上设置了 overflow 的值为 hidden 或 auto ，可以闭合浮动。

注意在 IE6 中还需要触发 hasLayout ，例如为父元素设置容器宽高或设置 zoom：1

我们可以使用overflow触发BFC（overflow不为visible触发），来触发独立渲染区域，并且利用其渲染规则中的第六条，计算其元素的高度包括浮动元素。

html结构

<body>
    <!-- 包裹元素 -->
    <div class="wrapper">
      <!-- 浮动元素 -->
      <div class="left">我是浮动元素</div>
      <div class="left">我也是浮动元素</div>
    </div>
</body>

CSS样式

.left{
    float: left;
}
.wrapper{
    overflow: hidden;
    zoom: 1; /* IE */
}

3.使用伪元素

原理很简单，使用伪元素在后面增加清除浮动的盒子。

在开发中比较常用的方法，拿来即用。

.clearfix {/* 触发 hasLayout */ zoom: 1; }
.clearfix:after {
    content: '';
    display: block;
    height: 0;
    clear: both;
    visibility: hidden; 
}

4.table+伪元素（最常用）

利用table表格的属性，当为空时不会占位，简化写法。

同时table即使没有高度也可以分割开两个盒子的margin，解决边距重叠的情况。

.clearfix:after,.clearfix:before {
    content: '';
    display: table
}
.clearfix:after{
	clear: both;
}
.clearfix {/* 兼容IE6,7触发 hasLayout */ zoom: 1; }
